Para Executar a aplicação será necessário:
1- instalar o node, pode ser feito no site: https://nodejs.org/en/
2- instalar o Angular cli, da seguinte forma: digitar no terminal 'npm install -g @angular/cli'
3- baixar o repositorio da aplicação
4- abrir a aplicação pelo terminal
5- Já na pasta da aplicação executar o comando 'ng serve'
5- Abrir o navegador e digitar o endereço: localhost:4200
